# Service: shortener-api (`api/`)

## Runtime

- **Runtime**: Bun latest stable
- **Framework**: Hono + TypeScript
- **ORM**: DrizzleORM (`drizzle-orm`, `drizzle-kit`)
- **Validation**: Zod
- **API Docs**: Swagger / OpenAPI via `@hono/zod-openapi`
- **Port**: 3000
- **Container**: `oven/bun:1` (build) → `gcr.io/distroless/base-debian12` (runtime)

## Endpoints

| Method   | Path             | Description                                                           |
| -------- | ---------------- | --------------------------------------------------------------------- |
| `POST`   | `/api/shorten`   | Accept `{ url }` → nanoid 6-char code → insert RDS → return short URL |
| `GET`    | `/:code`         | Cache lookup → miss: query RDS → push SQS event → return 301          |
| `DELETE` | `/api/url/:code` | Delete from RDS → invalidate cache entry                              |
| `GET`    | `/api/health`    | Ping postgres + cache → return status + latency_ms per service        |
| `GET`    | `/api/docs`      | Swagger UI                                                            |

import { z } from "zod";

const envSchema = z.object({
  DATABASE_URL: z.string().url(),
  SQS_URL: z.string().url(),
  BASE_URL: z.string().url(),
  PORT: z.coerce.number().default(3000),
  // Batch A

  DYNAMODB_REGION: z.string().optional(),
  // Batch B
  REDIS_URL: z.string().optional(),
});

## Cache Client Strategy

The `src/config/cache.ts` must detect which cache backend is active at startup
by checking which env vars are present:

```typescript
// Pseudo-logic — implement properly with types
if (env.DYNAMODB_TABLE && env.DYNAMODB_REGION) {
  // init DynamoDBClient from @aws-sdk/client-dynamodb
} else if (env.REDIS_URL) {
  // init Bun.Redis with env.REDIS_URL
} else {
  throw new Error("No cache backend configured — check SSM parameters");
}
```

`src/utils/cache.ts` must expose a unified interface regardless of backend:

```typescript
interface CacheClient {
  get(code: string): Promise<string | null>; // returns original_url or null
  set(code: string, originalUrl: string): Promise<void>; // TTL: 3600s
  del(code: string): Promise<void>;
}
```

## POST /api/shorten — Logic

```
1. Validate body: { url: string } via Zod
2. Generate 6-char code via nanoid
3. Check RDS: if code collision, regenerate (rare but handle it)
4. Insert into urls table: { code, original }
5. Return: { code, short_url: `${env.BASE_URL}/${code}`, original_url: url, created_at }
```

## GET /:code — Redirect Logic (Hot Path)

```
1. Check cache.get(code)
   HIT:
     2a. Push click event to SQS (non-blocking, fire and forget)
     3a. Return 301 → original_url
   MISS:
     2b. Query RDS: SELECT original FROM urls WHERE code = ?
         NOT FOUND → return 404
     3b. cache.set(code, original_url)
     4b. Push click event to SQS (non-blocking, fire and forget)
     5b. Return 301 → original_url
```

SQS push is non-blocking — do not await it in the redirect path.
Use `.catch(console.error)` to surface errors without blocking the response.

## DrizzleORM — urls Schema

```typescript
// src/database/schema/urls.ts
import { pgTable, serial, varchar, text, timestamp } from "drizzle-orm/pg-core";

export const urls = pgTable("urls", {
  id: serial("id").primaryKey(),
  code: varchar("code", { length: 10 }).unique().notNull(),
  original: text("original").notNull(),
  createdAt: timestamp("created_at", { withTimezone: true }).defaultNow().notNull(),
  expiresAt: timestamp("expires_at", { withTimezone: true }),
});
```

## DynamoDB Cache Operations (Batch A)

```typescript
// GET
const result = await client.send(
  new GetItemCommand({
    TableName: env.DYNAMODB_TABLE,
    Key: { code: { S: code } },
  }),
);
return result.Item?.original_url?.S ?? null;

// SET (with TTL)
await client.send(
  new PutItemCommand({
    TableName: env.DYNAMODB_TABLE,
    Item: {
      code: { S: code },
      original_url: { S: originalUrl },
      ttl: { N: String(Math.floor(Date.now() / 1000) + 3600) },
    },
  }),
);

// DEL
await client.send(
  new DeleteItemCommand({
    TableName: env.DYNAMODB_TABLE,
    Key: { code: { S: code } },
  }),
);
```

## Redis Cache Operations (Batch B)

```typescript
// Uses Bun.Redis (native — no npm package needed)
const redis = new Bun.Redis(env.REDIS_URL);

// GET
const value = await redis.get(`url:${code}`);

// SET with TTL
await redis.set(`url:${code}`, originalUrl, { ex: 3600 });

// DEL
await redis.del(`url:${code}`);
```
